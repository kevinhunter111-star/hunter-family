Private family dashboard hosted free on GitHub Pages.

- 💰 **Family Budget Dashboard** — TX & AR dual-state, bill import, tax estimate
- 📈 **Hunter Swing Trade System** — RSI, MACD, Bollinger Bands, AI-powered backtest

---

## Setup Instructions (One Time — ~10 minutes)

### Step 1 — Create a Free GitHub Account
1. Go to **github.com** and click **Sign up**
2. Choose a username (e.g. `hunterfamily` or `kevinhunter`)
3. Verify your email address

### Step 2 — Create a New Repository
1. Click the **+** button in the top right → **New repository**
2. Repository name: `hunter-family`
3. Set to **Public** (required for free GitHub Pages)
4. Click **Create repository**

### Step 3 — Upload Your Files
1. On the repository page, click **uploading an existing file**
2. Drag ALL 4 files into the upload area:
   - `index.html`
   - `budget.html`
   - `trading.html`
   - `README.md`
3. Scroll down and click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Click **Settings** tab in your repository
2. Click **Pages** in the left sidebar
3. Under **Source**, select **Deploy from a branch**
4. Under **Branch**, select **main** and click **Save**
5. Wait ~2 minutes

### Step 5 — Your Site is Live!
Your URL will be: `https://YOUR-USERNAME.github.io/hunter-family`

Share this URL with family. Default password: **hunter2025**
(Change it in the Settings tab after logging in)

## Add to Phone Home Screen

**iPhone:**
1. Open the URL in Safari
2. Tap the Share button (box with arrow)
3. Tap "Add to Home Screen"
4. Tap "Add"

**Android:**
1. Open the URL in Chrome
2. Tap the menu (⋮)
3. Tap "Add to Home screen"
